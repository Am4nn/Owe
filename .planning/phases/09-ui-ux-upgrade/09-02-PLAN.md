# Plan 09-02: Auth & Onboarding Flow

## Objective

Redesign all auth screens and add missing onboarding/forgot-password flows to match 4 Stitch screens: `splash_onboarding_owe_new_logo`, `sign_in_unified_google_button`, `sign_up_final_design`, `forgot_password_flow`.

> **Reference:** [CONTEXT.md §3.1–3.4](file:///c:/Aman/Coding-Bamzii/Web%20Development/Owe/artifacts/stitch_owe_design/CONTEXT.md)

### UI/UX Pro Max Strict Guidelines (apply to all tasks)
- **Interactions:** Use `Reanimated` spring scale interactions (150-300ms) or `activeOpacity={0.7}` on all pressable cards/buttons. No layout shifts on press.
- **Accessibility & Contrast:** Placeholder text must meet 4.5:1 contrast (use `#94A3B8`). Inputs must have a distinct focus state (e.g., 2px `border-brand-primary`). 
- **Icons:** Use ONLY Lucide SVG icons with a fixed 24x24 viewBox (`w-6 h-6`). No emoji icons allowed.
- **Glassmorphism:** Ensure dark tint overlays (`bg-white/5` to `bg-white/10`) with a visible 1px subtle border (`border-white/10`) so boundaries are visible.
