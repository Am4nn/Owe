# Plan 09-03: Dashboard & Profile

## Objective

Redesign the dashboard (populated + empty states), profile/settings, and add friend list/detail screens to match 5 Stitch screens: `dashboard_strict_dark_refinement`, `dashboard_empty_state`, `settings_profile_no_scrollbars`, `search_add_friends_no_scrollbars`, `friend_balance_sarah_miller`.

> **Reference:** [CONTEXT.md §3.5–3.6, §3.16–3.17, §3.24](file:///c:/Aman/Coding-Bamzii/Web%20Development/Owe/artifacts/stitch_owe_design/CONTEXT.md)

### UI/UX Pro Max Strict Guidelines (apply to all tasks)
- **Interactions:** Use `Reanimated` spring scale interactions (150-300ms) or `activeOpacity={0.7}` on all pressable cards/buttons. No layout shifts on press.
- **Accessibility & Contrast:** Placeholder and caption text must meet 4.5:1 contrast (use `#94A3B8`). 
- **Icons:** Use ONLY Lucide SVG icons with a fixed 24x24 viewBox (`w-6 h-6`). No emoji icons allowed.
- **Glassmorphism:** Ensure dark tint overlays (`bg-white/5` to `bg-white/10`) with a visible 1px subtle border (`border-white/10`) so boundaries are visible.

## Tasks

<task id="1" title="Convert Stack navigation to bottom tab navigator">

### What
Current `_layout.tsx` is a plain Stack navigator with an inline `COLORS` constant using old tokens (`#1C1C1E`, `#6C63FF`). Stitch designates a persistent bottom tab bar (Home, Activity, Groups, Profile) with a center FAB.

### How
- Install `@react-navigation/bottom-tabs` (if not already present — Expo Router supports Tabs layout)
- Convert `app/(app)/_layout.tsx` to use `Tabs` from `expo-router/tabs`
- Tab bar: solid `#0e1117` bg, top border `rgba(255,255,255,0.05)`, 4 tabs
  - Home (House icon), Activity (Clock), Groups (Users), Profile (User)
  - Active: `#7B5CF6` icon + label, Inactive: `#64748B`
  - Tab labels: 10px uppercase
- Center FAB: position absolute above tab bar, circular purple gradient, `+` icon, `glow-md` shadow
- Remove inline `COLORS` constant — imports from `theme.ts` instead
- Register new routes: `friends/index`, `friends/[id]`

</task>

<task id="2" title="Redesign dashboard — populated state">

### What
Current dashboard is a minimal FlatList of GroupCards with a BalanceSummaryBar. Stitch design has side-by-side balance cards, horizontal-scroll group cards with cover images, and a recent activity feed.

### How
**Layout (CONTEXT 3.5):**
- **Header:** Avatar (40px, pressable → profile) + "Owe" brand text left, search icon right
- **Balance cards (2, side-by-side row):** BalanceCard components
  - Left: "YOU ARE OWED" label (caption), teal amount, person count, green trend icon (IconContainer)
  - Right: "YOU OWE" label, red amount, person count, red trend icon
  - Both use GlassCard
- **Section:** SectionLabel "YOUR GROUPS" + "See All" action
- **Group cards (horizontal FlatList):** GroupCardHorizontal with cover image (160px height), group name, member count, balance — glass-card style
- **Section:** SectionLabel "RECENT ACTIVITY" + "See All" action
- **Activity items (vertical list):** ActivityItem rows with avatar, rich text, amount, timestamp, reaction badges

**Data:** Use existing `useGroups`, `useBalanceSummary`, plus new `useRecentActivity` hook (query recent expense events).

</task>
