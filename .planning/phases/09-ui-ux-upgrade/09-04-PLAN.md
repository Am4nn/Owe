# Plan 09-04: Group Detail & Expense Entry

## Objective

Redesign all group and expense screens to match 6 Stitch screens: `group_detail_dark_theme_refined`, `create_group`, `add_expense_strict_dark_theme`, `split_confirmation`, `expense_detail_dark_theme`, `expense_added_success`. Also restyle `settlement_success_strict_dark`.

> **Reference:** [CONTEXT.md §3.7–3.12, §3.18](file:///c:/Aman/Coding-Bamzii/Web%20Development/Owe/artifacts/stitch_owe_design/CONTEXT.md)

### UI/UX Pro Max Strict Guidelines (apply to all tasks)
- **Interactions:** Use `Reanimated` spring scale interactions on buttons. Cards should use `activeOpacity={0.7}`.
- **Accessibility & Contrast:** Form placeholders and labels must meet 4.5:1 contrast (use `#94A3B8`). Input borders should highlight on focus (`border-brand-primary`).
- **Icons:** Use ONLY Lucide SVG icons (`w-6 h-6`). No emoji icons allowed.
- **Glassmorphism:** Overlays/modals require strong blur (`expo-blur` `tint="dark"`). Base cards need subtle borders (`border-white/10`).
